### What is MyST?
MyST is a static website generator that we use to build our documentation pages.  That documentation is hosted using Github Pages.

### Adding Pages
Creating a new page can be done by creating a markdown document with a frontmatter header.
- [Markdown], a quick guide for writing markdown documents.
- [MyST Markdown], for MyST-specific features, including frontmatter.
